Validation

Catch dependency graph errors before startup.

Validation checks registrations without constructing service instances. That makes it safe for startup checks and CI smoke tests.

container = Container()
container.add_singleton(Settings)
container.add_transient(ApiClient)

container.assert_valid()

What validation checks

  • Missing constructor annotations.
  • Missing factory parameter annotations.
  • Missing registrations for required dependencies.
  • Cyclic dependencies.
  • Dependencies declared through injected property methods.

Collect all errors

errors = container.validate()

for error in errors:
    print(error)

Validation does not call constructors or factories. Runtime errors inside service constructors still happen when the service is resolved.
